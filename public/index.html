How to Use Greedy Algorithm Visualizer

Getting Started

  1. Choose an algorithm from the navigation bar
  2. Load a sample graph or create your own
  3. Configure algorithm parameters if needed
  4. Run the visualization to see the algorithm in action

Algorithms

Kruskal's Algorithm: Finds MST by sorting edges and avoiding cycles using Union-Find.

Prim's Algorithm: Builds MST by starting from a vertex and growing the tree.

Dijkstra's Algorithm: Finds shortest paths from a source vertex to all others.

Graph Format

Custom graphs should be in JSON format:

{
  "nodes": [
    {"id": "A", "label": "Node A", "x": 100, "y": 100},
    {"id": "B", "label": "Node B", "x": 200, "y": 150}
  ],
  "edges": [
    {"id": "A-B", "from": "A", "to": "B", "weight": 5}
  ]
}
